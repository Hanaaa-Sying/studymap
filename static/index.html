StudyMap

上传课程 PDF，自动生成可导入幕布的知识框架

使用说明

上传任意学习材料（课件、文献、书籍……），AI 提取结构后生成可导入幕布的 Markdown 文件。

📖 填充模式

提炼材料的大纲骨架，生成空框架。在后续阅读或学习过程中自主填充内容，逐步把这份材料内化为自己的知识结构。适合开始阅读前使用。

🎯 复习模式

全文扫描，把核心论点、话术和脉络全面梳理出来。适合完成阅读或学完一门课后，需要提纲挈领总结时使用，方便复盘或备考。
